## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/ferguevara2000/portafolio.git
```

Navigate to the project directory:

```bash
cd portafolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```
